# backend series to understand the backend

Dependencies - store or added also in the production
Dev-Dependencies = don't need to keep in the production. used only at the time of development

Some folders in the src:

**controllers -- to write the function which will handle the logic when user hits to a endpoint

**db -- to write the database connection logic in this folder

**middlewares -- to write different middleware functions in this folder

** routes -- to define different routes

** utils -- to define different functions which is acting like a different functionslity in project
